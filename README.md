I'm a Machine Learning Engineer with a focus on designing and deploying intelligent systems — from training neural networks to building the backend infrastructure that serves them at scale. My work spans the full ML lifecycle: data pipelines, model development, API integration, and production deployment.

- 🔬 &nbsp;**Research interest:** Improving large language model architectures and training efficiency
- ⚙️ &nbsp;**Engineering background:** Distributed systems, REST APIs, and ML-serving infrastructure with Django & Flask
- 🤝 &nbsp;**Open to collaborate on:** ML research projects, LLM applications, and backend systems
- 💬 &nbsp;**Ask me about:** Neural networks, model training pipelines, or Python-based ML workflows
